    1. What is Semantic HTML? 

        Semantic HTML is using HTML while making use of the meaning of information in the website opposed to just defining the presentation.

    7. What is an id? 

        An id is an attribute to specifiy a unique id for an element, the value of every id must be unique within the document.  Used to manipulate a specific element in the page.

    8. What elements can I add an id to? 

        Any element that would have specific styling or require manipulation from JavaScript that is not shared with another element.  Could also use it as a "bookmark" to travel to a section of the page faster.

    9. How many times can I use the same id on a page? 

        Just once.

    1. What is the difference between an inline element and a block element?

        Inline elements don't force a new line to begin the flow.  Block elements will typically cause a line break unless changed in the CSS.

    2. What happens when an element is positioned absolutely? 

        Depending on where the element is placed, it will either be relative to the nearest positioned ancestor or if it does not have a positioned ancestor it will use the document body and scroll along the page.
